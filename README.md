# 📖 速讀詩篇 (Speed Psalms App)

這是一個簡單且輕量的網頁 App，旨在幫助讀者在一個月內讀完一遍詩篇（共 150 篇）。

## 🚀 功能特點
- **自動計算**：每天打開 App，自動根據日期計算當天應讀的 5 篇詩篇。
- **手機優化**：採用 App 式 UI 設計，支援 iPhone 與 Android 「加入主畫面」全螢幕使用。
- **特殊邏輯**：
  - 1 到 30 號：每天顯示 5 篇（日期 d, d+30, d+60, d+90, d+120）。
  - 31 號：專讀詩篇第 119 篇（最長篇章）。

## 🛠 如何安裝到手機？
1. 用手機瀏覽器打開 [你的網頁連結]。
2. **iPhone (Safari)**：點擊「分享」>「加入主畫面」。
3. **Android (Chrome)**：點擊選單 >「安裝應用程式」或「新增至主畫面」。
